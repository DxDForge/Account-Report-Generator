# Bella Ciao - Daily PDF Report Generator

## What This App Does

This is a **simple calculator app** that generates PDF reports for your daily business. 

**NO DATABASE** - Just fill the form, click "Generate PDF", and you get a ready-to-share PDF report!

## Features

✅ Single form with all fields  
✅ Auto-calculates totals, closing balances, achievement rates  
✅ Add customer reviews inline  
✅ **Generate PDF with one click**  
✅ **Preview and Save/Share PDF**  
✅ No data storage - fresh start each time  

## Perfect For

- Daily WhatsApp/Telegram group updates
- Partner reports
- Quick calculations
- PDF documentation

## How to Use

1. **Open the app**
2. **Select date** (tap the green card at top)
3. **Fill in the numbers:**
   - Opening balances
   - Today's sales
   - Expenses
   - Target
   - Improvement notes
4. **Add reviews** (optional - tap "Add Customer Review")
5. **Click "GENERATE PDF REPORT"**
6. **Save or Share** the PDF!

## Installation

```bash
# Extract and install
unzip bella_ciao_pdf_generator.zip
cd bella_ciao_pdf_generator
flutter pub get
flutter run
```
